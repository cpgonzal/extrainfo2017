¿Qué es Jupyter?¶

  • El Jupyter Notebook es un entorno interactivo web que permite ejecutar código de ciertos programas:
  • El alumno utiliza sólo un navegador (no es necesario instalar ningún programa) y puede introducir instrucciones a través de un cuaderno de trabajo (notebook) muy sencillo de manejar.
  • Cada vez que se ejecuta una instrucción, ésta se procesa por el servidor y envía el resultado al notebook.
  • El entorno permite que el alumno tenga un espacio de trabajo personal donde puede crear nuevos notebooks en los que trabajar y realizar las diferentes tareas .
  • Al finalizar, el alumno puede descargar su notebook y enviarlo al profesor para su evaluación.

Proyecto de innovación docente de la ESIT¶

  • La dirección de la Escuela Superior de Ingeniería y Tecnología presentó este año el proyecto:

  • El Centro de Cálculo de la Escuela instala una plataforma (jupyterhub) para el uso de los notebooks de Jupyter
  • Tanto los profesores como alumnos pueden acceder cada uno a un espacio de trabajo donde crear y desarrollar las actividades con los notebooks

Funcionamiento y aspecto¶

¿Qué aspecto presenta el notebook de jupyter?¶

  • El notebook tiene el siguiente aspecto:
  • Permite introducir código (operaciones de cálculo, gráficos, etc..), etiquetas, ecuaciones $\LaTeX$, ... así como descargar el notebook en múltiples formatos
  • Los alumnos pueden editar y ejecutar en vivo el código
  • Se puede crear una versión estática del notebook para que otros la puedan ver
  • Tiene un enorme interés por lo que supone para la Investigación reproducible

def answer_five():
    df_expenses = df.filter("actividad LIKE '%CLUBS%'" 
                    or "actividad LIKE '%SALAS FIESTA%'" 
                    or "actividad LIKE '%PUBS%'" 
                    or "actividad LIKE '%SDISCOTECAS%'"
                    or "actividad LIKE '%BARES%'")
    df_expenses_mapped = df_expenses.rdd.map(lambda x: (x.nombre,x.importe))
    df_expenses_reduced = df_expenses_mapped.reduceByKey(lambda x,y:x+y)
    df_expenses_top=df_expenses_reduced.map(lambda x:(x[1], x[0])).sortByKey(ascending=False).take(10)
    
    return pd.DataFrame([ (x[1],x[0]) for x in df_expenses_top ] ,columns=['name','total_amount'])
